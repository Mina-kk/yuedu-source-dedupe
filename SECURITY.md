# 安全说明

## 报告安全问题

请不要在公开 Issue 中披露可被利用的漏洞、私人书源、Cookie、Token、签名密钥或服务器凭据。

发现安全问题时，请通过 GitHub 仓库所有者公开资料中提供的私密联系方式报告，并附上：

- 受影响版本
- 复现步骤
- 影响说明
- 建议修复方式（如有）

## APK 校验

仅从本项目 GitHub Releases 下载 APK，并核对发布页提供的 SHA-256。Android 安装包必须由发布者签名；来源不明或摘要不一致的文件不应安装。

## 第三方内容

应用可访问用户输入的 URL 和第三方书源页面。请勿导入不可信来源；恶意或异常数据可能导致错误结果、流量消耗或第三方隐私风险。
